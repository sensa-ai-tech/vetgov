## 一、我們為什麼要做這件事

2026 年 7 月 1 日起,《人用藥品用於犬貓及非經濟動物之使用管理辦法》強制施行。

新制要求人用藥品須由藥廠完成「動保用藥」品保登錄,才能合法販售給動物醫院。聽起來合理。但 ——

**701 項公告處方藥,目前只有 144 項完成登錄(20.5%)。**

剩下的八成,包含急救針劑、強心劑、利尿劑、消炎藥、抗生素,甚至醫用氧氣。動物用藥僅占人用藥廠營收 1–2%,藥廠普遍缺乏配合誘因。法規從 2024 年 2 月公告至今已兩年,登錄率仍未過半。**這不是時間問題,是制度設計違反市場邏輯。**

真正的問題不在這個管理辦法,而在《藥事法第 50 條》—— 五十年來,該條從未將動物醫院納入合法購藥對象。獸醫師長期處於「合法使用、非法購買」的窒礙狀況。2012 年的點滴斷供事件、2022 年的 Palladia 三聯停產事件,都是這個結構性問題的代表性後果。

**本站訴求(三大):**
1. **修正藥事法第 50 條** —— 將動物醫院納入合法購藥對象(終結五十年灰色地帶)
2. **負面表列取代正面表列** —— 參照美國 FDA 模式,僅列禁用品項
3. **暫緩管理辦法 7/1 強制上路** —— 在修法完成前

**5 月底為法案排入立法院衛環委員會審議的死線。**

## 二、站點定位

一個以**獸醫師第一線視角**為出發點、具有**渲染力但不煽情**的公民議題追蹤平台：

- **頭版宣言**：「我們是台灣的獸醫師。我們不反對處方制度。我們反對的是——一個沒有配套、沒有溝通、沒有問過第一線能不能做到的新制。」
- **五大訴求**：暫緩上路試辦、電子處方系統、罰則歸屬、正式公聽會、偏鄉預算
- **第一線處境**：六個執業現場的真實情境（一診 8 分鐘、處方流通、成本吸收、偏鄉守護、罰則打誰、我們不是敵人）
- **政策時間線**：每日自動抓取媒體與公告，補充第一線觀點註記
- **行動呼籲**：致農業部長與防檢署長的公開信

**非目標**：
- 不做個人攻擊、不煽動對立
- 不提供法律或獸醫專業建議
- 不代表「全體獸醫師」或任何公會發言
- 不接受商業贊助、不嵌入廣告、不追蹤使用者

## 三、技術棧

| 層 | 選型 | 理由 |
|---|---|---|
| 爬取 | `feedparser` + `httpx` | 純 Python，無瀏覽器依賴 |
| 分類 | 關鍵字 regex | 零 API 成本，可離線跑 |
| 萃取 | Claude Haiku 4.5（選用） | 便宜，每日 ~20 則內容 <NT$5 |
| 儲存 | SQLite | 單檔可 git commit，零維運 |
| 前端 | 原生 HTML/CSS/JS | 無 build step，秒部署 |
| 排程 | GitHub Actions | 免費 2000 min/月 |
| 部署 | Cloudflare Pages | 免費、CDN 快、支援自訂網域 |

## 四、成本

- **Anthropic API**：~NT$150/月（併入現有預算）
- **網域**：NT$300–800/年（維護者自購）
- **主機**：NT$0（Cloudflare Pages free tier）
- **維護**：約每月 30–60 分鐘（爬蟲源變動、人工補充重大事件）

## 五、內容結構

1. **頂部警示條**：倒數 + 快速導覽至五大訴求
2. **開場宣言 Hero**：對農業部的致辭 + 強制上路倒數
3. **立場聲明**：「一份來自第一線的聲明」
4. **五大訴求**：暫緩試辦 / 電子處方 / 罰則歸屬 / 公聽會 / 偏鄉預算
5. **第一線處境**：六個真實情境卡片
6. **政策時間線**：自動追蹤 + 人工補充
7. **資料來源透明度**：所有監測來源公開列出
8. **行動呼籲**：致農業部長的公開信式收尾
9. **關於**：我們是誰、我們相信什麼、運作原理、錯誤回報

## 六、里程碑

| 里程碑 | 狀態 | 交付 |
|---|---|---|
| M1 資料層可運作 | ✅ | scraper + classifier + storage 單元測試通過 |
| M2 離線模式可產出時間線 | ✅ | `cli build-site` 能從 seed 產出 `data.json` |
| M3 第一線告白版 Hero + 訴求 + 現場聲音 | ✅ | `site/index.html` 渲染完整 |
| M4 GitHub Actions 跑過一次 | ⚪ Day 2 | 推到 GitHub + 綁網域 |
| M5 累積 30 天真實資料 | ⚪ Day 30 | 驗證 pipeline 穩定性 |
| M6 獸醫社群擴散 | ⚪ Day 14 | 至少 3 位執業獸醫具名背書 |

## 七、倫理與透明度

1. **資料來源公開**：`site/` 顯示所有監測的 RSS URL
2. **LLM 使用標註**：每個事件標註 `人工整理` / `AI 萃取`
3. **原始連結保留**：每個事件都連回原始新聞來源
4. **不追蹤個人**：只記錄公職人員、組織發言人的公開言論
5. **錯誤更正**：歡迎 PR / issue 更正事實錯誤
6. **立場透明**：本站有明確訴求（呼籲暫緩 + 配套），不偽裝中立，但每一項指控都必須有可查證的來源

## 八、給下一位維護者的話

這不是一個情緒發洩的平台，也不是一個反改革的堡壘。這是我們對主管機關的請求——**把我們當成一起把制度做好的夥伴，而不是上路後才收到公告的執行工具。**

如果有一天農業部真的公布了電子處方系統、試辦區計畫、配套預算，這個站會第一時間把訴求換成「肯定與監督」，繼續追蹤實施成效。我們不是要永遠反對，我們是要等一個對得起動物、對得起飼主、也對得起第一線的制度。
